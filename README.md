Este é o meu projeto mais complexo e completo até o momento. Desenvolvido para simular a experiência de um aplicativo de pedidos para uma lanchonete real.

- **💻 Repositório:** [AlexOnn1/leaozinho-prototipo-app](https://github.com/AlexOnn1/leaozinho-prototipo-app)
